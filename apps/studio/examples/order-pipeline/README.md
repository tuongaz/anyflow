# Order Pipeline — SeeFlow Example

A four-stage e-commerce pipeline (Order → Inventory → Payment → Fulfillment) running in a single Bun process. Hit **Play** on the canvas and watch each service light up in sequence as events flow through the chain.

## Run

```bash
bun start
```

SeeFlow registers this example automatically on first launch. Open the studio at `http://localhost:4321` and click **Play** on the **POST /orders** node.
